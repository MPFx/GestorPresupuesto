# Gestor de Presupuesto Personal

## Versión
**1.0**

## Descripción
Aplicación de consola para gestionar finanzas personales. Permite registrar ingresos y gastos, establecer metas de ahorro, categorizar transacciones y generar reportes financieros.

## Alcance

### ✅ Qué hace
- Registrar ingresos con categorías
- Registrar gastos con categorías
- Categorizar transacciones (Comida, Transporte, Salario, etc.)
- Ver balance actual (ingresos - gastos)
- Establecer metas de ahorro
- Mostrar alertas cuando se excede el límite por categoría
- Generar reportes en múltiples formatos (Texto, Gráfico, PDF simulado)

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real

## Tipos de Transacción

| Tipo | Descripción |
|------|-------------|
| Ingreso | Dinero que entra (Salario, Regalo, Venta) |
| Gasto | Dinero que sale (Comida, Transporte, Salud) |

## Reportes Disponibles

| Tipo | Descripción |
|------|-------------|
| Texto | Reporte detallado en formato texto |
| Gráfico | Reporte con gráficos ASCII |
| PDF | Reporte simulado en formato PDF |

## Estructura del Presupuesto

| Mes | Ingresos | Gastos | Balance |
|-----|----------|--------|---------|
| Configurable | Suma de ingresos | Suma de gastos | Ingresos - Gastos |
